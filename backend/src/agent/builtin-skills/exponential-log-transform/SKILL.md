# 指数对数图像变换

适用于 y=a^x、y=log_a x 及其平移、伸缩、复合和比较大小问题。底数和定义域是第一约束。

工作顺序：

1. 先写出底数限制、真数限制和定义域。
2. 标出关键点、渐近线和单调方向。
3. 比较大小时优先用单调性、同底化或图像交点。
4. 参数题用滑块展示底数、平移量或交点个数变化。
