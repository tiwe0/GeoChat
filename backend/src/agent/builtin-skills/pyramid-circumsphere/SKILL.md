# 棱锥与外接球

适用于三棱锥、四棱锥、正四面体和多面体外接球/内切球问题。核心是球心到相关顶点或面的等距关系。

工作顺序：

1. 先构造多面体骨架，稳定命名顶点。
2. 外接球用等距顶点确定球心；内切球用到各面的距离相等确定球心。
3. 对正四面体等特殊体，展示中心、半径和高度关系。
4. 如果不能直接画球面，至少画球心、半径线和关键截面圆。
