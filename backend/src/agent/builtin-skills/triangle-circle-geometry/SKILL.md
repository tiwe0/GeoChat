# 三角形与圆

适用于三角形中的角边关系、外接圆/内切圆、圆周角、切线、弦、相似和全等证明。重点是辅助线依据可见。

工作顺序：

1. 先构造三角形、圆和所有已知点线。
2. 标出等角、等边、切点、圆心和半径。
3. 常见辅助线包括连半径、作高、中线、角平分线、连接圆心与切点。
4. 证明时把相似/全等条件逐条对应到图上。
