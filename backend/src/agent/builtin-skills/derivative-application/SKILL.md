# 导数及其应用

适用于切线、导函数、单调区间、极值最值、函数零点、恒成立和参数讨论。核心是把 f、f'、切线和临界点放在同一视图中解释。

工作顺序：

1. 明确函数定义域，求导前先处理参数和不可导点。
2. 同时画 f(x) 和 f'(x)，用 f' 的符号解释 f 的增减。
3. 极值/最值题标出临界点、端点和函数值；切线题标出切点与斜率。
4. 参数题寻找切线重合、重根、极值穿越水平线或区间端点变化。

禁忌：不要只给导数表；必须把临界点和原函数图像对应起来。
