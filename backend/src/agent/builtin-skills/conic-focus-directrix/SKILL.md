# 圆锥曲线焦点准线

适用于椭圆、双曲线、抛物线的标准方程、焦点、准线、离心率、切线、弦长和轨迹问题。

工作顺序：

1. 先确定曲线类型、标准方程和坐标轴方向。
2. 标出焦点、顶点、准线、渐近线或对称轴。
3. 直线与曲线问题同时展示交点、弦中点和斜率。
4. 轨迹题用动点或参数滑块展示生成关系。
