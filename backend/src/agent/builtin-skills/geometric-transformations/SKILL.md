# 图形变换

适用于平移、旋转、轴对称、中心对称、位似、相似变换、尺规作图和测量。目标是展示变换前后对象的一一对应关系。

工作顺序：

1. 识别变换类型、变换中心/轴/向量/角度/比例。
2. 先构造原图，再构造像；用同色或虚实线表现对应点、对应边和轨迹。
3. 对旋转和轴对称，必须显示中心、角度或对称轴；对位似，必须显示位似中心和比例。
4. 对作图题，用步骤说明每一条辅助线或圆弧的几何依据。

常用 GeoGebra 方向：Reflect、Rotate、Translate、Dilate、Segment、Circle、Angle、Distance、Locus。
