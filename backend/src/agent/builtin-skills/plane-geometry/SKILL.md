# 平面几何

优先把题目转成可验证的几何对象：点、线、圆、角、长度、交点、垂线、平行线和轨迹。不要只给文字证明。

工作顺序：

1. 提取已知点线圆、目标结论和隐含关系。
2. 先构造公共底图，再补辅助线。
3. 对相似、全等、圆周角、切线、垂径定理等关系，用颜色和标签区分依据。
4. 如果题目是选择题，把每个选项当成独立 scenario 验证。

常用 GeoGebra 方向：Line、Segment、Circle、Intersect、PerpendicularLine、ParallelLine、Angle、Distance、Midpoint。
