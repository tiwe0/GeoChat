# 多项式函数

主路径必须把代数条件落到坐标平面：函数图像、零点、交点、切线、对称轴、顶点、参数滑块或关键水平线。

关注点：

- 二次函数优先展示顶点、对称轴、判别式对应的交点数量。
- 三次/高次函数优先展示零点、局部极值和导数符号区间。
- 参数题尽量使用滑块和动态标记展示临界值变化。
- 选择题用不同 scenario 展示每个选项成立或失败的视觉证据。
