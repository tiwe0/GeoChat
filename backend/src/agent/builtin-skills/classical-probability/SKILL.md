# 古典概率

适用于有限等可能样本空间、抽取、掷骰、摸球、排列组合、条件概率和独立事件。重点是先列清事件。

工作顺序：

1. 判断是否等可能；不等可能不能直接用古典概型。
2. 列样本空间和目标事件，必要时用树状图或表格。
3. 排列组合题说明是否有序、是否放回、是否重复。
4. 条件概率要先缩小样本空间。
