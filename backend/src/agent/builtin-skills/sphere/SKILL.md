# 球

球题先定位球心、半径和约束面。若命令参考没有 GeoGebra 5 可用的球体命令，不要臆造命令，改用中心、半径、截面圆和辅助线表达关键关系。

关注点：

- 外接球：球心到所有相关顶点距离相等。
- 截面：截面圆半径、球半径、球心到截面距离组成直角三角形。
- 与多面体结合：先构造多面体骨架，再标出球心候选和等距关系。
