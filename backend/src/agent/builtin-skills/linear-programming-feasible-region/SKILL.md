# 可行域与目标函数

适用于二元一次不等式组、可行域顶点、目标函数最值和实际规划题。必须把目标函数等值线画出来。

工作顺序：

1. 每个约束先画边界直线，再用测试点判断半平面。
2. 求可行域顶点并标注坐标。
3. 目标函数写成等值线，用平移解释最优位置。
4. 实际题要把变量含义、非负约束和单位说明清楚。
