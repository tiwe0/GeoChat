# 尺规作图

适用于要求作图步骤、轨迹依据或只允许几何构造的题目。每一步都要能对应到圆弧、交点、垂线或角平分线。

工作顺序：

1. 明确目标对象和允许的基础对象。
2. 用圆、交点、垂直平分线、角平分线等基本构造生成对象。
3. 保留必要辅助线，隐藏无关辅助对象。
4. 说明每一步成立的几何依据。
