# 因式分解与公式

适用于提公因式、平方差、完全平方、十字相乘、分组分解和配方法。重点是先识别结构，再选择变形。

工作顺序：

1. 先检查公因式、次数、项数和特殊系数。
2. 判断是否符合平方差、完全平方或二次三项式结构。
3. 分解后回乘验证，避免符号错误。
4. 配方题要说明加减同一个量的原因。
