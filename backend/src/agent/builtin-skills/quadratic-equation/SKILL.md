# 一元二次方程

适用于求根、判别根的个数、根与系数关系、参数方程和二次函数零点问题。代数解和图像零点要互相对应。

工作顺序：

1. 整理为 ax^2+bx+c=0，并说明 a≠0。
2. 根据题目选择因式分解、配方、公式法或根与系数关系。
3. 参数题优先分析判别式、重根或根所在区间。
4. 需要可视化时画 y=ax^2+bx+c 和 x 轴交点。
