# 二次函数

适用于 y=ax^2+bx+c、顶点式、交点式、最值和参数图像问题。必须展示开口、顶点、对称轴和与坐标轴交点。

工作顺序：

1. 判断表达式形态，必要时转为顶点式或交点式。
2. 标出顶点、对称轴、开口方向和零点个数。
3. 最值题说明定义域；闭区间最值要比较端点和顶点。
4. 参数题用滑块或临界图像展示交点个数变化。
