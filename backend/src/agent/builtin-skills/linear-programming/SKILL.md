# 线性规划

必须画出约束边界、半平面方向、可行域和目标函数等值线。最终答案要绑定到可行域顶点或边界。

工作顺序：

1. 把每个不等式转成边界直线和取侧测试点。
2. 标出可行域顶点。
3. 绘制目标函数等值线，必要时用参数滑块平移。
4. 对最值，逐点或沿边验证，不只给代数结论。
