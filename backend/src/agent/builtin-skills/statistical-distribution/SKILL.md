# 统计图表与分布

适用于样本数据整理、频数/频率分布、条形图、折线图、直方图、箱线图、平均数、中位数、众数和方差。

工作顺序：

1. 先判断数据类型：分类、离散数值或连续区间。
2. 选择合适图表，不用一种图表硬套所有题。
3. 计算代表数时说明它反映的是集中趋势还是离散程度。
4. 样本估计总体时说明随机性和误差来源。
